學會更改 Hexo 網站框架的主題樣式

Posted on 2022-04-15 In 架站 Symbols count in article: 719 Reading time ≈ 1 mins.

在完成網站基本的架設與部署到 Github 上面後，現在要學習怎麼把 Hexo 的主題更改成自己喜歡的樣式，而在 Hexo 的官方網站上也有提供許多網友製作的主題可以提供大家下載，那就準備動手幫自己的網站換上新的衣服吧！

大綱

  • 選擇自己喜歡的主題樣式
  • 更改根目錄的 _config.yml 檔案
  • 測試結果

選擇自己喜歡的主題樣式

可以直接進到 Hexo 的網頁，找自己喜歡的主題樣式，目前大多數的人是使用的是 Next 主題，而今天的示範內容也會是以 Next 作為範例，如果是選擇其他主題的人，也只要跟著步驟操作，更換檔案名稱就可以了！

首先需要下載主題樣式的檔案，可以直接用 Git 的方式 clone 到自己的本機端
打開終端機 cd 到網站根目錄裡面的 theme 資料夾並輸入下方文字

如果不是選擇 Next 主題，只要更改 clone 後面的網址就可以了。

更改根目錄的 _config.yml 檔案

已經完成主題樣式的檔案下載後，就可著手更改網站端的文件內容，可以在 VS code 或任何文字編輯器中打開網站根目錄的 _config.yml 檔案，
找到

theme: landscape（預設主題）

更改後面的 landscape 為 next (或者其他主題名稱，要記得跟 theme 資料夾中的名稱相同)，這樣就算是完成主題的更改了！
